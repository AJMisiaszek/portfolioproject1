  1. Computer Literacy: The knowledge and understanding of basic computer fundamentals.
  2. Computer: A programmable, electronic device that accepts data input, performs processing operations on that data, and outputs and stores the results.
  3. Input: The process of entering data into a computer; can also refer to the data itself
  4. Processing: Performing operations on data that has been input into a computer to convert that input to output.
  5. Output: The process of presenting the results of processing; can also refer to the results themselves
  6. Storage: The operation of saving data, programs, or output for future use.
  7. Communications: The transmission of data from one device to another
  8. Data: Raw, unorganized facts
  9. Information: Data that has been processed into a meaningful form.
  10. Hardware: The physical parts of a computer system, such as the keyboard, monitor, printer, and so forth
  11. Software: The instructions, also called computer programs, that are used to tell a computer what it should do
  12. Operating System: The main component of system software that enables a computer to operate, manage its activities and the resources under its control, run application programs, and interface with the user
  13. Boot: To start up a computer.
  14. Windows Desktop: The background work area displayed on the screen for computers running Microsoft Windows
  15. Application Software: The background work area displayed on the screen for computers running Microsoft Windows
  16. Embedded Computer: A tiny computer embedded in a product and designed to perform specific tasks or functions for that product.
  17. Mobile Device: A very small device that has built-in computing or Internet capability
  18. Smartphone: A mobile device based on a mobile phone that includes Internet capabilities and can run mobile apps
  19. Media Tablet: A mobile device based on a mobile phone that includes Internet capabilities and can run mobile apps
  20. Personal Computer (P.C.): A type of computer based on a microprocessor and designed to be used by one person at a time; also called a microcomputer.
